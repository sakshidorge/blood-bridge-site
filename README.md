# 🩸 BloodBridge — Saving Lives Together

A complete blood donation management website built with HTML, CSS, and JavaScript.

## 🌐 Live Demo
Deployed on Vercel — [bloodbridge.vercel.app](https://bloodbridge.vercel.app)

## ✨ Features

- **Home Page** — Hero section, live blood stock, blood group chips, stats
- **Find Donor** — Search donors by blood group, city, and state
- **Donor Registration** — Full form with eligibility criteria
- **Blood Statistics** — Charts (distribution, stock, monthly trends, state-wise)
- **🚨 Emergency Request** — Urgent blood request with alert system
- **Hospital Directory** — 12+ partner hospitals with emergency contacts
- **Donor Tips** — 9 tips for before/during/after donation
- **Admin Dashboard** — Manage donors, stock (+/−), emergencies, hospitals
- **Contact Page** — Contact form + emergency hotline

## 🩸 Blood Groups Supported
`A+` `A−` `B+` `B−` `O+` `O−` `AB+` `AB−`

## 🚀 Deploy to Vercel

### Option 1: One-click via Vercel Dashboard
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Click **Deploy** — no configuration needed!

### Option 2: Vercel CLI
```bash
npm i -g vercel
vercel
```
